Class: Task

Task(id, content, completed, creationDate, lastUpdateDate)

Stores information about task.

It stores task ID, its content, status (is it completed or not), date of creation and date of last update (if task wasn't updated the date of creation equals to last update date).

Constructor

new Task(id, content, completed, creationDate, lastUpdateDate)

Creates task instance.
Parameters:
Name Type Description
id string task id.
content string content of the task.
completed boolean task status (is it completed or not).
creationDate Date date when task was created.
lastUpdateDate Date date when task was last updated (corresponds to {@param creationDate} if task was not updated.
